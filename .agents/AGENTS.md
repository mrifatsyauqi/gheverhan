# Project Rules

## Git Workflow
- Setiap kali sebuah fitur, perbaikan bug, atau instruksi selesai dikerjakan, kamu WAJIB secara otomatis menjalankan perintah `git add .`, `git commit -m "[deskripsi perubahan]"`, dan `git push` ke repositori GitHub.
- Jangan menunggu user meminta untuk melakukan push. Jadikan ini sebagai standar prosedur operasi (SOP) di akhir setiap pekerjaan.
